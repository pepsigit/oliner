NGUI - Label


  • Label

    • 在Hierarchy窗口中，选中UI Root
    • 在Scene视图中，在UI Root中右键单击 : Create -> Label
    • 属性 :
      • Font Atlas :
        • NGUI : 使用NGUI的字体集
        • Unity : 使用系统字体，如ttf
      • Font Size : 字体大小，受限于Widget下Size属性
      • Text : 文本内容
        • 可以设置[颜色值]内容[-]是自己自定义变色，颜色值会和Color Tint颜色进行融合
        • 文本加粗[b][/b]
        • 文本斜体[i]italic[/i]
        • 文字加下划线[u]underline[/u]
        • 直线穿过文字[s]strikethrough[/s]
        • 让文字在左下角或右上角显示[sub]sub[/sub] or [sup]sup[/sup]
        • 文字超链接显示格式[url=http:xxx]clickable hyperlinks[/url](详情方法见下面)
      • Overflow :
        • ShrinkContent : 字体大小自适应控件大小
        • ClampContent : 裁剪
        • ResizeFreely : 控件大小自适应字体大小
        • ResizeHeight : 控件高度自适应字体需要的高度
      • Alignment : 对齐方式
        • Justified : 自动填充宽度(只对英文有效)
      • Gridient : 颜色渐变
      • Space : 字间距、行间距
      • BBCode : 文本属性的编码识别

    • 文本超链接

      • 需要为Label添加Box Collider : 右键单击文本Attach -> Box Collider

      • 为Label添加脚本 :

        • void OnClick()
        • {
        • UILabel lbl = GetComponent<UILabel>();
        • string url = lbl.GetUrlAtPosition(UICamera.lastWorldPosition);
        • Application.OpenURL(url);
        • }

      • 若只想点击只有超链接文字大小的区域才会响应，请设置Box Collider的Center、Size，并将Widget下的auto-adjust to match取消勾选

    • 使用图片

      • 修改UIFont，参考Arimo20

    • 动态文字

      • content : UILabel，并有脚本Typewriter Effect
        • Chars Per Second : 每秒显示多少个字符
        • Fade In Time : 字符的淡入时间
        • Delay on Period : 延迟期
        • Delay on New Line : 开始新行延迟时间
        • Keep Full Dimentions : 保持全部尺寸，一开始就显示最大尺寸
      • bg : UISprite，content的孩子，作为背景
      • UILabel的Overflow设置为ResizeHeight
      • bg的Anchors的Type设置为Unified，Execute为OnUpdate，Target为content即可
