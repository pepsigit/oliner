Mesh Filter

网格过滤器

  • 属性

    • Mesh : 引用将要被渲染的网格。

  • mesh:

    • mesh像是值传递。

    • 使用mesh会多次克隆对象，对具体的object拷贝，改变此mesh只改变当前对象。

    • 改变mesh中的数据，改变的是真实数据。

      • 运行时：改变数据，只会改变当前物体的形状，不影响基于此mesh的其他物体，退出程序时，数据恢复。( Unity5.3已验证 )
      • 编辑时：改变数据，退出unity后，数据无法恢复。( Unity5.3会改变形状，但是会报错，Instantiating mesh due to calling MeshFilter.mesh during edit mode. This will leak meshes. Please use MeshFilter.sharedMesh instead. )

  • sharedMesh :

    • sharedMesh像是引用传递。

    • sharedMesh只有一份，改变此sharedMesh会改变所有。

    • 改变sharedMesh中的数据，改变的是缓存中的数据。

      • 运行时：改变数据，会改变所有基于此mesh的物体形状，退出程序时，数据无法恢复。退出unity后，数据恢复。( Unity5.3已验证 )
      • 编辑时：改变数据，退出unity后，数据恢复。( Unity5.3已验证 )


Mesh Render

网格渲染器

  • 属性

    • Cast Shadows : 是否开启投射阴影。

    • Recieve Shadows : 是否接受阴影投射。

    • Use Light Probes : 是否使用灯光探头。

  • material:

    • 每次更换属性，unity会自动new一份新的material。直到Application.LoadLevel()或者Resources.UnloadUnusedAssets()时才会释放内存。

  • sharedMaterial:

    • 共享材质，无论如何操作材质的属性（颜色、shader），内存中只会占有一份。
