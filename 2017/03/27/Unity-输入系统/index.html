[Unity]输入系统

  • 时间：2017-03-27
    • Unity

  • 默认输入轴：

    • Horizontal、Vertical映射到a、d、w、s。
    • Fire1、Fire2、Fire3映射到left control、left alt、left shift （ mac-unity ）。
    • Mouse X、Mouse Y映射到鼠标移动增量。


  • 添加新输入轴：


  • 虚拟轴：

    • Input.GetAixs( @param: string )
    • @return ( -1 ~ 1 )
    • Input.GetAixsRaw( @param: string )
    • @return ( -1 || 0 || 1 )


  • 鼠标输入：

    • GetMouseButton( @param : index )：鼠标按下，只会执行一次。
    • GetMouseButtonDown( @param : index )：鼠标一直按下状态。
    • GetMouseButtonUp( @param : index )：鼠标抬起，只会执行一次。
    • @param : 0 : 左键 || 1 : 右键 || 2 : 中键
    • @return bool


  • 键盘输入：

    • GetButton( @param : string )：按键按下，只会执行一次。
    • GetButtonDown( @param : string )：按键一直按下状态。
    • GetButtonUp( @param : string )：按键抬起，只会执行一次。
    • GetKey( @param : string || KeyCode )：按键按下，只会执行一次。
    • GetKeyDown( @param : string || KeyCode )：按键一直按下状态。
    • GetKeyUp( @param : string || KeyCode )：按键抬起，只会执行一次。
    • @return bool


  • 触摸：

    • Touch.GetTouch( 0 ).phase : 返回最近一帧的触摸状态。
    • Input.touches : 返回上一帧所有的触摸状态的对象列表（readonly）。
    • Touch.deltaPosition : 距离上次改变的距离增量。
    • TouchPhase :

      • Began : 手指已触摸屏幕。

      • Moved : 手指在屏幕上移动。

      • Stationary : 手指触摸屏幕，但未移动。

      • Endedn : 手指离开屏幕，触摸的最后状态。

      • Canceled : 系统取消跟踪触摸，如用户把屏幕放到他脸上或超过五个接触同时发生。这是一个触摸的最后状态。
