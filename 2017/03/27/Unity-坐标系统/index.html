[Unity]坐标系统

  • 时间：2017-03-27
    • Unity

  • World Space ( 世界坐标系 )

    • 场景中的物体，以世界坐标系显示在场景中。transform.position可以获得该位置坐标。

  • Screen Space （ 屏幕坐标系 ）

    • 这是一个以像素来定义的坐标系。
    • 屏幕的左下角为（ 0，0 ）；右上角为（ Screen.width，Screen.height ）
    • 鼠标、触摸位置属于屏幕坐标（Input.mousePosition || Input.GetTouch(0).position）
    • Screen.width = Camera.pixelWidth Screen.height = Camera.pixelHeight

  • ViewPort Space （ 视口坐标系 ）

    • 视口坐标是标准的坐标系，相对于相机。
    • 屏幕的左下角为（ 0，0 ）；右上角为（ 1，1 ）

  • GUI Space （ 绘制GUI坐标系 ）

    • 相似于屏幕坐标系。
    • 屏幕的左上角为（ 0，0 ）；右下角为（ Screen.width，Screen.height ）


  • 坐标系之间转换

    • 世界坐标 -> 屏幕坐标

      • Camera.main.WorldToScreenPoint( transform.position )

      • @return Vector3

    • 屏幕坐标 -> 世界坐标

      • Camera.main.ScreenToWorldPoint( new Vector3( 100, 100, camera.nearClipPlane ) )

      • @return Vector3

    • 屏幕坐标 -> 视口坐标

      • Camera.main.ScreenToViewportPoint( Input.GetTouch( 0 ).position )

      • @return Vector3

    • 视口坐标 -> 屏幕坐标

      • Camera.main.ViewportToScreenPoint( new Vector3( (0.0~1.0f), (0.0~1.0f), 0.0f ) )

      • @return Vector3

    • 世界坐标 -> 视口坐标

      • Camera.main.WorldToViewportPoint( transform.position )

      • @return Vector3

    • 视口坐标 -> 世界坐标

      • Camera.main.ViewportToWorldPoint( new Vector3( (0.0~1.0f), (0.0~1.0f), camera.nearClipPlane ) )

      • @return Vector3
